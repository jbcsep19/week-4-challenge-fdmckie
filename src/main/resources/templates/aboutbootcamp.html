About Java Bootcamp





  The Java Bootcamp is an 8-week intensive course that not only involves core Java coding but also the following:

  • HTML 5 (Hypertext Markup Language)
  • CSS 3 (Cascading Style Sheets)
  • Java Script
  • Security coding
  • SQL (mySQL)

  With the use of Intellij software, it allows Java, as the controller, to call pages, written in other programming languages, to display web pages.

  The purpose of the Java Bootcamp is to enhance one's programming skills. Hopefully, this will help with future job placement.
