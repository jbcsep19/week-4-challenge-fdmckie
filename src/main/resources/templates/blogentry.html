  Title:      What Is core Java?      Date:      08/21/2019

  Core Java refers to Java SE. It includes the Java Language, the Java Virtual Machine (JVM) and Java Development Kit (JDK). The JDK contains the compiler,   some tools and a library. The library is called the Java API (Application Programming Interface). It consists of classes developed by Oracle/Sun that go with   Java. Core Java is used for general purpose programming.

  Title:      What Is GitHub?      Date:      08/21/2019

  GitHub is a web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality    of Git as well as adding its own features. Unlike Git, which is strictly a command-line tool, GitHub provides a web-based graphical interface and desktop as   well as mobile integration. It also provides access control and several collaboration features such as wikis, task management, and bug tracking and feature   requests for every project.

  Title:      What Is Git?      Date:      08/21/2019

  Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.   Git commands are used to "push" programming code into a GitHub repository.

  Title:      What Is Inheritance?      Date:      08/21/2019

  The idea of inheritance is simple but powerful: When you want to create a new class and there is already a class that includes some of the code that you   want, you can derive your new class from the existing class. A subclass inherits all the members (fields, methods, and nested classes) from its superclass.

  Title:      Algorithm      Date:      08/21/2019

  An algorithm represents the ordered sequence of steps required to solve a problem. The algorithm gets translated to a programming language, such as   Java. In Java, the Java compiler will translate the program into instructions the computer can execute. This produces the required output from the given   input.

  Title:      Pseudocode      Date:      08/21/2019

  Pseudocode is a human-readable representation of an algorithm. It is not written the specific syntax of any particular programming language. However, a   programmer can easily translate pseudocode into a programming language, like Java, because the pseudocode is the algorithm plus any considerations for   making it into a program.
